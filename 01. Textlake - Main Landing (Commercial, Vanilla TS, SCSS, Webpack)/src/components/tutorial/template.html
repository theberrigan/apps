How to create an account

Thank you for your interest in Textlake!

To create an account:

  1. Navigate to https://tsm.textlake.com.
  2. After the page is loaded, click on "Create Account" tab.
  3. Type your email address and password in the corresponding fields. Password should be at least 8-character long and include numbers, upper and lower case characters.
  4. Click "Create Account" button. If the selected password is strong enough, you will be automatically logged in to the application.
  5. To proceed, you have to read and accept "Terms and Conditions". If you are agree with the terms, confirm it by checking the box and clicking on "Accept" button.
  6. Next, select your subscription plan:
    • You can choose "Free" or "Business" plan.
    • If you decide to start with "Free" plan, you can always upgrade it to "Business" using "Settings → Subscriptions" menu.
    • If you decided to start with "Business" plan to have full benefits immediately, the application will ask you to select the number of users and provide credit/debit card information. You will be charged monthly based on your selections.
  7. To confirm your email, click on "Verify Email" button on the top of the page.
    • "Email verification" pop-up will be displayed; you have to open your email inbox in a new browser tab, copy the confirmation code and paste it into "Verification code" field.
    • If the confirmation code is correct, you will be promptly notified.
    • If there is a problem with the confirmation code, you have several login attempts before the code becomes invalid. If you need to regenerate the code, please close the pop-up window and click on "Verify Email" again to receive a new code.
  8. Congratulations! Now you have a verified Textlake account!

Notes:
As an alternative, you can create a Textlake account using one of your social media accounts by clicking on "Facebook" or "Google" buttons on the login page.

If you have problems or questions, please email contact@textlake.com.

How to change user preferences

To change user preferences:

  1. Click on the arrow next to your email at the top of the main menu (if it is collapsed, click on "Burger" button first).
  2. Click on "Profile" menu item.
  3. On Profile page, you can customize the following settings:
    1. "Language" - Textlake user interface language. Currently, the application supports English, Polish, Russian, and Spanish.
    2. "Display Datetime Format" - defines date display format for the application.
    3. "Input Datetime Format" - defines date input format for the application.
    4. "Status Colors" - please read "How to add colors" for more information.

How to set up a company

Company management requires "Company - Edit" role to be assigned to your account. If you do not have the required role, please first read Matrix of Roles.

To set up a company:

  1. Click on "Settings → Company Profile".
  2. Fill in the fields on Company Profile page; note that all fields from Company Profile page are visible to public and shown on quotes, offers, invoices, emails, etc.
    1. "Preferred Language" - the preferred language for your company. It is used by default for communication with clients unless there is a different preferred language specified by a client.
    2. "Timezone" - the time zone used for displaying date and time information.
  3. "Save" the changes.
  4. Click "Settings → Currencies".
  5. On Currencies page:
    1. Select the primary currency by clicking "Make Primary" button. The primary currency is used by your company for accounting and tax reporting, so choose it thoughtfully. While it is possible to adjust this setting in the future, we highly discourage any changes to keep reporting consistent.
    2. Select all currencies which you plan to accept by checking "Enabled" box and setting exchange rates.
    3. To see exchange rate history, click on "View History" button.
  6. "Save" the changes.
  7. Click "Settings → Taxes".
  8. On Taxes page:
    1. Click "Add" button to create a new entry which will be displayed on Offers page. For example, Tax Name = "Poland 23%" and Rate = "23.0000".
    2. You can have multiple tax entries per company.
  9. "Save" the changes.

How to add a service

Service management requires "Service - Edit" role to be assigned to your account. If you do not have the required role, please first read Matrix of Roles.

To add a service:

  1. Click on "Settings → Services" tab.
  2. Click on "Create" button.
  3. Fill the following fields:
    • Name - name of the service as you want is to be seen by clients.
    • Short name - alternative name of the service for internal use.
    • Direction - direction of the translation (for example, English to Spanish).
    • Price - per unit cost of the service (in primary currency) to be paid by a client.
    • Rate - the default rate is "Basic"; to add custom rates, please read #13.
    • Unit - work measurement unit (for example - page, 1800 chars, 1 hour, etc.).
  4. Click "Save" to accept the changes.

To clone an existing service:

  1. Click "Settings → Services".
  2. Select one or many services to be cloned by checking the box next to the service name. Note that only services under "Basic" rate can be cloned.
  3. Click "Clone" button at the bottom of the page.
  4. You will see a list of services to be cloned. Select the required custom rate for each service and then click "OK" to apply the changes.

To delete a service:

  1. Click "Settings → Services".
  2. Click "Delete" button next to the service to be deleted. Note that only services with custom rates can be deleted.

How to create an offer

Offer management requires "Offer - Edit" role to be assigned to your account. If you do not have the required role, please first read Matrix of Roles.

An offer can be created in two different ways - from scratch or based on an email.

To create an offer from scratch:

  1. Click "Offers" tab.
  2. Click "Create" button to see a blank offer.
  3. Next, you have to select the existing client or create a new client record:
    1. On "Details" tab under "Client & Contact" section, click "Find" button and see the client selection menu.
    2. Type the name of a client to see potential matches.
    3. If an existing client is found, click on the client's name and press "Select" button.
    4. If there is no matching record, click "Create" and "OK" buttons to create a new one.
  4. Fill out "Person", "Email", "Phone", and "Fax" fields with contact information.
  5. Add required information into "Miscellaneous", "Delivery", and "Translation" sections.
  6. Select "Currency" and "Tax" applicable to the client at the bottom of the page.
  7. Click "Add Service" button to add services requested by the client.
    1. On "Services" page, select one or more services. Please use filters to narrow down search results.
    2. Check the service(s) required and click "Update" button.
  8. You should see a list of services displayed on "Details" tab. For each service, update the following fields:
    1. Units - defines the volume of work to be done as requested by the client.
    2. Billable Units - defines the volume of work to be billed to the client.
    3. Ratio and Discount - provides additional adjustments for billing.
    4. Billable - detects whether the service is visible and billable to the client./li>
  9. Click on "Document" tab to add documents to be translated.
    1. Click on "Documents" section or drag and drop files directly into this section.
    2. As soon as the files are uploaded, you will be able to add public and private comments to each document.
  10. Next step is to assign documents with services.
    1. Click on "Details" tab.
    2. Each service has "Select documents" menu. Click on it and select the document/s to be associated./li>
  11. On "Shipping" tab, fill out shipping information optional).
  12. Click "Save" to create an offer.

To create an offer from an email:

  1. Click "Mailbox" and select required email. If no mailboxes is available, please first read How to attach a mailbox.
  2. Click "Create new offer" button at the top of the list.
    1. If the email has attachments, you will be prompted to select the attachment/s to be uploaded.
    2. After the attachments are selected, click "Create Offer" button.
  3. Next steps are similar to the guide above except the documents are added automatically, and contact information is filled in for existing clients.

How to add a client

Client management requires "Offer - Edit" role to be assigned to your account. If you do not have the required role, please first read Matrix of Roles.

How to add a client from an offer is described in "How to create an offer" tutorial.

To add a client:

  1. Click "Clients" menu.
  2. Click "Create" button.
  3. On "General" tab:
    1. Fill in "Name" or "Legal Name" fields.
    2. Select "Preferred Language" if it is different from your company's preferred language. This selection will be used for all your communication with the client.
    3. Fill in other required information.
  4. On "Billing" tab, fill in the client's financial information if available.
  5. Click "Save".
  6. New tabs will be displayed:
    1. Contacts - directory of contacts on the client's side
      1. To create a new contact, click "Add contact" button.
      2. Fill in the requested information.
      3. Check "Make primary" if you want to make a contact primary - it will be automatically selected for every new offer for the client. Only one contact can be primary.
      4. Check "Inactive" if you want to mark a contact as inactive.
      5. Click "OK".
    2. Rates - please refer to "How to use custom rates" tutorial for additional information.
    3. Balance - financial balance of the client.
      1. Click "All" button to see all transactions made by the client.
      2. Click "Credit" button to see all credit translations made by the client.
      3. Click "Debit" button to see all debit translations made by the client.
      4. Click "Create Transaction" and populate "Amount", "Currency", and "Comment" fields to adjust the client's balance. Click "OK" to create the transaction.
    4. Advanced - allows deletion and restoration of a client.
      1. Click "Delete Client" button to delete a client. The client will be instantly marked as deleted.
      2. Click "Restore Client" button to restore a client. The client will be instantly restored.
      3. Check "Show deleted clients" and click "Search" button to see deleted clients on client list ("Clients" menu).
  7. Click "Save" to apply the changes.

How to add a translator

Translator management requires "Translator - Edit" role to be assigned to your account. If you do not have the required role, please first read Matrix of Roles.

To add a translator:

  1. Click on "Translators" tab.
  2. Click on "Add" button.
  3. On "General" tab:
    1. Fill in "First Name" and "Last Name" or "Legal Name" of the translator.
    2. Provide other required information.
    3. Make sure that "Active" checkbox is enabled.
  4. On "Financial" tab, add the information required for tax reporting purposes.
  5. On "Services" tab:
    1. Click "Add Service" button.
    2. Select "Company Service" which will be the default translator's service. You can use "Filter Service" to narrow the list down. If you do not see any company services, please read How to add a service.
    3. Add one or many areas of the translator's proficiency in "Subject Area".
    4. Select "Translation type".
    5. Provide "Price" and "Currency" per unit of work to be paid to the translator.
    6. Check "Notary" if the translator is offering notarizing services.
    7. Check "Extra Service" if the service is not performed by the translator on a regular basis but he/she is willing to help with it sometimes.
    8. Click "OK".
    9. Repeat the steps to add more services.
  6. Click "Save" to apply the changes.

How to create a quote

To create a quote manually:

  1. Open an offer.
  2. Click "Show quote" button to see the quote preview page.
  3. Click "Copy" button to copy the quote.
  4. Open your preferred email client, create a new email or reply to an existing thread, and paste the quote.

To create a quote automatically:

  1. Open an offer.
  2. Click "Save & Send" button.
  3. On "Send a quote" pop-up screen, add all the required information and click "Save & Send" button again.
  4. If you see an error message saying that your email is not verified, please click the link and open your preferred email client. Once you receive an email from Amazon Web Services, please click on the link received. This verification step is required by AWS to prevent fraud and email forging.
  5. The quote has been sent to your client.

How to add a mailbox

Mailbox management requires "Mailbox - Edit" role to be assigned to your account. If you do not have the required role, please first read Matrix of Roles.

To add a new mailbox:

  1. Click "Settings → Mailboxes".
  2. Click "Create" button.
  3. Add "Email" address to be used.
  4. Add "Password" for the above email address - it will be encrypted and securely stored.
  5. Add "Host" and "Port" of your email server.
    1. The most common parameters are:
      1. Gmail - HOST imap.gmail.com, PORT 993, SSL true
      2. Yahoo - HOST imap.mail.yahoo.com, PORT 993, SSL true
      3. AOL - HOST imap.aol.com, PORT 993, SSL true
      4. MSN - HOST imap-mail.outlook.com, PORT 993, SSL true
      5. Office 365 - HOST outlook.office365.com, PORT 993, SSL true
      6. Outlook.com - HOST imap-mail.outlook.com, PORT 993, SSL true
      7. Hotmail.com - HOST imap-mail.outlook.com, PORT 993, SSL true
      8. Live.com - HOST imap-mail.outlook.com, PORT 993, SSL true
      9. Yahoo! - HOST imap.mail.yahoo.com, PORT 993, SSL true
    2. If you are using a custom email server, please contact your system administrator.
  6. Keep "Protocol" as IMAP.
  7. Select whether a secured connection should be used during communication with your email server.
  8. Click "Test" button to verify the connection.
  9. If the test is successful, click "Save" button to apply the changes.

NOTE:
Some users reported that occasionally they were unable to establish successful connection despite the correct credentials were provided. It is a know issue with major email providers such as Gmail, Yahoo, etc. The providers consider IMAP protocol usage is less secure than OAuth 2.0 authentication flow. We are planning to add support of OAuth 2.0 in the future but if you experienced similar problem please let us know as soon as possible - it will help us prioritize development of new features.

Meanwhile to solve connectivity problem please do the following steps:

  • GMAIL
    1. Open https://myaccount.google.com/lesssecureapps?pli=1 in a new browser tab.
    2. Activate "Allow less secure apps".
    3. Open https://accounts.google.com/b/0/DisplayUnlockCaptcha in a new browser tab.
    4. Click "Continue" button.
    5. You can now connect your Gmail account to Textlake.

  • YAHOO!
    1. Log into Yahoo go to "Settings → Account Info".
    2. Then go to the "Account security" section and enter your username and password.
    3. Once you’re in the Account security section activate the "Allow apps that use less secure sign in".
    4. You can now connect your Yahoo Mail account to Textlake.

How to invite a new user

User management requires "Company - Edit" role to be assigned to your account. If you do not have the required role, please first read Matrix of Roles.

To invite a new user:

  1. Click "Settings → Invitations" tab.
  2. Click "Add" button.
  3. Provide "Email" of the user to be invited.
  4. Select "Roles" to be assigned to the user.
  5. Click "Save" to apply the changes.
  6. Ask user to go to https://tsm.textlake.com and create an account using the email provided on the step #3.
  7. If the user already has an account, please ask user to:
    1. Open "Profile" (click on the arrow next to email address field at the top left part of the screen).

    3. Click "Switch" button.

How to create a project

Project management requires "Project - Edit" role to be assigned to your account. If you do not have the required role, please first read Matrix of Roles.

  1. Open the offer accepted by your client.
  2. Click "Create Project" button.
  3. If you have "Project Coordinator - Edit" role, a pop-up window will suggest choosing a coordinator who will be assigned to the project. Please select one or keep it unassigned.
  4. Click "OK" to confirm the selection.
  5. After a new project is created, you will be redirected to the project screen.
  6. On "Details" tab, set "Deadline" for the project by clicking on calendar button.
  7. Next, for each service assign a translator responsible for the project deliverable/s.
    1. Translators can be assigned by clicking on "Assign translator to service" button (if one person is handling the entire service) or by clicking on "Assign translator to attachment" (if one person is assigned to a specific document).
    2. "Add Assignment" screen will be displayed.
    3. Fill in the required fields:
      1. "Begin" - indication of where the assignment starts (for example, page 5).
      2. "End" - indication of where the assignment ends (for example, page 10).
      3. "Units" - number of work units to be completed.
      4. "Deadline" - the date when the assignment has to be completed.
      5. "Accepted" - the date when the assignment has been accepted by the translator.
      6. "Comment" - any additional comments.
      7. "Reclaimed amount" - amount to be reclaimed from the translator if the quality requirements are not satisfied.
      8. "Reclamation comment" - reasons for the reclamation.
    4. Select a translator from the list below and click "OK". If you do not see any translators, please read #8.
    5. Repeat the steps if necessary.
  8. Click "Save" to apply changes.
  9. When the final version of the translated document is provided by the translator, click on "Upload" button and select the document to attach. You will be able to download it any time later by clicking on "Download" button.

How to use custom rates

User management requires "Company - Edit" and "Offer - Edit" roles to be assigned to your account. If you do not have the required role, please first read Matrix of Roles.

To add a custom rate:

  1. Click "Settings → Rates".
  2. Click "Add" button.
  3. Provide the following values:
    1. Name - name of the rate.
    2. Description - description of the rate for internal use.
    3. Enabled - check if the rate is active.
    4. Global - check if the rate is applicable to all clients.
  4. Click "Save" to apply the changes.
  5. Click "Clients" on the left hand side menu.
  6. Select the client you would like to assign the rate to.
  7. Click on "Rates" tab.
  8. The list of all active rates will be displayed. The global rates will be checked by default.
  9. Check the rate you would like to assign.
  10. Click "Save".

How to collect payments online

User management requires "Company - Edit" role to be assigned to your account. If you do not have the required role, please first read Matrix of Roles.

  1. Click "Settings → Payment Providers".
  2. On the payment providers screen, click on tab representing the provider you would like to use.
  3. "Stripe" tab:
    1. Click on "Connect with Stripe" button.
    2. If you already have a Stripe account, click on "Already have a Stripe account? Sign in" link and login with your credentials.
    3. If you do not have a Stripe account, then fill in the form and click "Authorize access to this account" button.
    4. After successful sign-in or sing-up, you will be redirected back to Stripe configuration page.
    5. Check "Enabled" to enable the provider.
    6. Click "Save" to apply the changes.
  4. SOFORT tab (available for Business subscription only).
    1. Add API Key, Customer ID, and Project ID values received from your SOFORT account. (Image below shows an example how to get API Key).
    2. Check "Enabled" to enable the provider.
    3. Click "Save" to apply the changes.
  5. Next time you send a quote to a client by clicking on "Save and Send" button on Offer page, your client will be able to pay online from the email or PDF file received.

How to update your subscription

Subscription management requires "Subscription - Edit" role to be assigned to your account. If you do not have the required role, please first read Matrix of Roles.

To update your subscription:

  1. Click "Settings → Subscriptions".
  2. Click "Change" button next to the Plan name."Change Plan" page will be displayed.
  3. Select the required plan and click on "Choose" button.
  4. Select the number of active users you would like to have.
  5. Fill in card holder name, card number (16 digits), expiration date, CVV, and ZIP code (optional for debit cards). The provided card will be charged every month with the amount displayed.
  6. Click "Submit" to confirm the selections.

To update the number of active users:

  1. Click "Settings → Subscriptions".
  2. Click "Change" button next to Users number.
  3. Select new number and click "Save" button.
  4. Confirm your active status by clicking "OK" button.
  5. If you have a paid plan, the new monthly amount will be displayed below the funding source.

To update debit/credit card:

  1. Click "Settings → Subscriptions".
  2. Click "Change" button next to Funding Source.
  3. Fill in new card information - card holder name, card number (16 digits), expiration date, CVV, and ZIP code (optional for debit cards).
  4. Click "Submit" to confirm the changes.

How to view reports

Company management requires "Project - Edit" role to be assigned to your account. If you do not have the required role, please first read Matrix of Roles.

To view reports:

  1. Click "Reports" on the left menu and select a report.
  2. "Translators" - summary of work grouped by translator.
    1. Set up your filters by entering:
      1. Translator name - (optional) name of the translator.
      2. Project - (optional) project id.
      3. Project status - current project status. Pick "Any" to skip filtering by status.
      4. Project created since - projects created after this date.
      5. Project completed before - projects completed before this date. To be completed, a project has to have Closed or Canceled status.
      6. Comment - (optional) full or partial comment from the statement record.
    2. Click "Search".
    3. The report by translator will be displayed.
      1. Project - project id.
      2. Status - project status.
      3. Amount - amount to be paid to the translator in the primary currency.
      4. Payment Id - unique id created when a payment is made.
      5. Paid check box - indicates if the payment has been made.
      6. Comment - comment related to the payment.
    4. If any change is made to the report, click "Save" button at the bottom of the page.
  3. "Projects" - summary of work grouped by project. The workflow is the same as for Translators report (see step #2).
  4. "Coordinators" - summary of work grouped by coordinator.
    1. Set up your filters by entering:
      1. Coordinator - name of the coordinator.
      2. Offer Created Since - offers created after this date.
      3. Offer Created Before - offers created before this date.
      4. Project Created Since - projects created after this date.
      5. Project Created Before - projects created before this date.
      6. Project Completed Since - projects finished after this date. To be completed, a project has to have Closed or Canceled status.
      7. Project Completed Before - projects finished before this date. To be completed, a project has to have Closed or Canceled status.
      8. Project status - current project status. Pick "Any" to skip filtering by status.
      9. Offer status - current offer status. Pick "Any" to skip filtering by status.
    2. Click "Search".
    3. The report by coordinator will be displayed. Total Gross and Total Net are the amounts in the primary currency calculated based on filter values.
  5. "Clients" - summary of work grouped by client
    1. Set up your filters by entering:
      1. Name - (optional) name of the client
      2. Project status - current project status. Pick "Any" to skip filtering by status.
      3. Project Created Since - (optional) projects created after this date.
      4. Project Created Before - (optional) projects created before this date.
      5. Project Completed Since - (optional) projects finished after this date. To be completed, a project has to have Closed or Canceled status.
      6. Project Completed Before - projects finished before this date. To be completed, a project has to have Closed or Canceled status.
      7. Offer/Project - (optional) offer or project id.
      8. Invoiced - (optional) indicates if the invoice has been sent to the client.
    2. Click "Search".
    3. The report by client will be displayed.
      1. You can click on the arrow to display client information.
      2. Comment - editable comment related to the payment.
      3. Invoice - invoice number.
      4. Send - indicates if the invoice has been sent to the client.
    4. If any change is made to the report, click "Save" button at the bottom of the page.

How to add colors

Textlake application supports color customization for Offer, Project, and Translator records.

  1. Offers and Projects
    1. Click on the arrow next to your email at the top of the main menu (if it is collapsed, click on "Burger" button first).
    2. Click on "Profile" menu item.
    3. On Profile page,click on a rectangle next to offer status name.
    4. Pick the desired color from the color picker.
    5. Click "Save" to apply the changes.
    6. Now if you click on Offers or Projects menu, the rows will be highlighted with the selected colors.
  2. Translators
    1. Make sure that "Translator - Edit" role is assigned to your account.
    2. Click "Translators" menu.
    3. Click "Edit" button next to the translator.
    4. On "General" tab, click on a rectangle with a label "Color".
    5. Pick the desired color from 20 options available.
    6. Click "Save" to apply the changes. The new color will be visible by all company users.

Notifications

Users will be informed about important events by getting personal or group notifications. The feature is available even when a user is offline.

  1. To view a notification, click on "Bell" icon at the top right corner.
  2. Most notifications are interactive (for example, if you get a message "Offer 2019-1 received 100 USD", you can click on it to access Offer # 2019-1).

Task manager

Task manager informs company users about approaching project deadlines.

  1. To view tasks, click on "Tasks" button at the top right corner.
  2. Each task has:
    1. Color
      1. Blue - the deadline is between 6 and 24 hours from now.
      2. Yellow - the deadline is between 2 and 6 hours from now.
      3. Red - the deadline is less than in 2 hours from now.
    2. Status - Open or Closed.
    3. Message - description of work to be completed.
    4. Coordinator - email of the task responsible.
    5. Time - date and time when the task was created or updated.
  3. When a Project is finished, the corresponding Task will be automatically updated as "Closed".
