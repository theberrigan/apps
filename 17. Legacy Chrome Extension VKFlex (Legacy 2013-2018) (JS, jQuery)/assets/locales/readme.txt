В этой папке находятся файлы с локализацией расширения.
Вы можете перевести расширение на любой язык и отправить локализацию автору.

ВНИМАНИЕ! Есть некоторые правила форматирования файлов локализации:

1. Файлы должны быть в кодировке UTF-8 (без BOM).

2. В них запрещено использование любых комментариев.
   // Такой комментарий запрещён
   /* Такой тоже запрещён */

3. Каждый ключ и значение должны быть заключены в двойные кавычки (").
